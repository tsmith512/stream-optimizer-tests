The Journal

Lots of customers have videos like these in storage somewhere. Migrating them to Stream and delivering them along with an HLS player library is cumbersome.

Outputs

Optimized

Just the default options returns a 640w at 2.2MB.

Still Frames

For use in galleries, search results, etc.

"scale-down" scales to maintain aspect ratio, containing the entire input video within the provided output dimensions.

"contain" is similar, but will also upscale to maximize the fit.

"cover" scales to maintain aspect ratio to completely cover the output dimensions, then crops any overflow.

"crop" is still in development
